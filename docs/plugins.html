Extra Plugins

jmpress.js also comes with extra plugins located in the /plugins folder that you might choose to use. To load the plugin simply add a script tag including the plugin js file.

Duration

For automatically changing steps after a given duration. Can also display a jQuery UI progress bar indicating how long until the change will occur. Adds the following object to the jmpress.js config:

duration: {
	defaultValue: -1 // Duration in ms, default is -1 which mean inactive
	,defaultAction: "next" // Event to fire, ie: next or prev
	,barSelector: undefined // jQuery UI progress bar element
	,barProperty: "width" // Width of progress bar
	,barPropertyStart: "0" // Starting position of progress bar
	,barPropertyEnd: "100%" // Ending position of progress bar
}

Writing a Plugin

To extend jmpress.js, simply create a new plugin js file and wrap all your plugin code within a closure. Then to alter or add new configuration settings:

$.jmpress("defaults").mynewsetting = '';

Now you can hook in functionality by adding callbacks:

$.jmpress("setActive", function( step, eventData ) {
	// Do something new when a step is activated
}

Take a look at the core plugins in jmpress.js or the extra plugins in the /plugins folder for examples.
