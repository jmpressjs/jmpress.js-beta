Options

stepSelector

Default: '.step'
The jQuery selector to specify each step.

containerClass

A class name to set on the container.

canvasClass

A class name to set on the canvas.

areaClass

notSupportedClass

Default: 'not-supported'
Class name to remove on root element if jmpress.js is not supported.

loadedClass

Default: 'loaded'
Class name to set on each step that has been AJAX loaded.

fullscreen

Default: true
Whether jmpress.js should run in full screen mode or in a container.

animation

Set the CSS animation values for transitions between slides.
